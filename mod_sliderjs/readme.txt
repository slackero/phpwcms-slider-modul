Installation:
-------------

Unzip the downloaded file and copy the complete folder 'mod_jqsliderjs' to your phpwcms
installation

The module will install itself and add no tables to your phpwcms database.
To do that go to the modules section an click the new 'SliderJS' menu item.

That's all!
